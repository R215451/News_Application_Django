## 📌 Overview  

A dynamic **News Web Application** built using **Python Django** that fetches and displays the latest news through API integration.  
The project focuses on clean backend logic, secure configuration, and responsive frontend design.

> 🎯 Focus: Backend Logic • API Integration • Clean UI • Deployment Ready

---

## 🚀 Features  

- 📰 **News API Integration**  
  - Fetches real-time news using external API  
  - Displays categorized and updated content  

- 🔐 **Environment Variables**  
  - API keys and sensitive data managed securely using `.env`  

- 🧩 **Django App Structure**  
  - Modular structure using `startapp`  
  - Clean separation of logic and templates  

- 🔗 **Page Navigation**  
  - Uses Django `url` tags for smooth routing between pages  

- 🎨 **Responsive UI**  
  - Designed using W3.CSS  
  - Mobile-friendly and clean layout  

- 🌐 **Deployment**  
  - Hosted on Render  
  - Live project accessible via public URL

## ⚙️ Installation & Setup  

```bash
# Clone the repository
git clone https://github.com/R215451/News_Application_Django.git

# Navigate to project folder
cd live_news

# Create virtual environment
python -m venv venv

# Activate environment
venv\Scripts\activate   # (Windows)

# Install dependencies
pip install -r requirements.txt

# Run server
python manage.py runserver
